## 9. 使用规则

1. **新术语必须先在本文定义**，再在其他文档中引用
2. **代码中的命名应与本表英文一致**：类名、变量名、API 路径均使用英文术语
3. **UI 中的标签使用中文术语**：按钮、标题、提示文字使用本表中文定义
4. **术语变更需同步更新**：修改本文后，须检查 Doc 7（领域模型）和 Doc 8（系统架构）的一致性
5. **中英混排**：中文与英文/数字间加空格（如 `Agent 调度` 而非 `Agent调度`）

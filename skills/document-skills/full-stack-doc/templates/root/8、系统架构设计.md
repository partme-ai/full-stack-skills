```mermaid
sequenceDiagram

    participant API as API Server
    participant CMD as Commander
    participant Agent as Agent Worker
    participant CLI as OpenCLI
    participant Platform as 电商平台
    participant DB as Database

    User->>API: POST /tasks/create
    API->>CMD: 创建编排任务
```

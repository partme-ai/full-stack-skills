# document-skills

文档与图表相关技能组。Marketplace 插件 `document-skills` 还引用 `docx` / `pptx` / `pdf` / `xlsx`（部分目录待补）。

## 本组目录内技能（已存在 `SKILL.md`）

| 技能 | 说明 |
|------|------|
| [api-doc-generator](api-doc-generator/) | API 文档生成 |
| [doc-coauthoring](doc-coauthoring/) | 文档协作 |
| [full-stack-doc](full-stack-doc/) | **产品文档库标准布局**（根 11 + 版本 7 + 模块 PRD/Stitch/UI）；模板见 `full-stack-doc/templates/` 下 `root/`、`version/`、`module/`、`delivery/` |
| [mermaid](mermaid/) | Mermaid 图表 |
| [plantuml](plantuml/) | PlantUML |
| [processon-mindmap](processon-mindmap/) | ProcessOn 思维导图 |

安装：`/plugin install document-skills@full-stack-skills`

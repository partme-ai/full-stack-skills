## When to use this skill

Use this skill whenever the user wants to:
- 用 Django 创建 Web 应用、ORM、admin、视图与路由
- 配置项目、应用、模型、迁移与部署

## How to use this skill

1. **起步**：django-admin startproject；startapp、settings、INSTALLED_APPS。
2. **核心**：models、makemigrations/migrate；views、urls、templates；admin 注册。
3. **参考**：https://docs.djangoproject.com/

## Best Practices

- 一应用一职责；模型规范化与索引。
- 视图瘦、逻辑在 service 或 model 层；安全与 CSRF、权限。

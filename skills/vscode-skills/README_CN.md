# VS Code Skills

这套技能旨在帮助 AI Agent 辅助零基础用户从头开始构建 Visual Studio Code 插件。

## 概览

本插件提供了 VS Code 插件开发的完整工作流指导，包括：

1.  **项目初始化**：一键生成项目脚手架。
2.  **功能实现**：添加命令、Webview 界面、侧边栏等。
3.  **打包发布**：构建 `.vsix` 安装包。

## 技能列表

*   **`vscode-project-init`**：使用官方生成器创建新的插件项目。
*   **`vscode-feature-command`**：向插件清单和代码中添加新命令。
*   **`vscode-feature-webview`**：使用 Webview 实现自定义 UI 界面。
*   **`vscode-deploy-package`**：将插件打包发布。

## 使用方法

将此插件加载到您的 AI Agent（如 Claude Code）中，然后使用自然语言请求开发任务。

**示例：**
*   "创建一个名为 'todo-list' 的 VS Code 插件。"
*   "添加一个 'Show Todo' 命令，打开一个 Webview 界面。"
*   "帮我打包这个插件。"
